# 信息展示设计指南

本指南说明了如何有效地展示 AI 开发过程中的各种信息。

## 信息类型与展示方式

### 1. 系统提示词 (System Prompt)
- **目的**：展示 AI 角色设定和任务指令
- **展示位置**：详细视图的标签页中
- **视觉设计**：
  - 使用紫色/紫罗兰色调区分
  - 代码块样式，等宽字体
  - 可折叠/展开
- **交互**：允许复制，只读

### 2. 用户输入 (User Input)
- **目的**：展示发送给 AI 的原始输入
- **展示位置**：详细视图标签页
- **视觉设计**：
  - 使用靛蓝色调
  - 与系统提示词区分开
  - 支持语法高亮（如果是代码）

### 3. AI 输出 (AI Output)
- **目的**：展示 AI 的最终响应
- **展示位置**：默认视图，标签页默认激活
- **视觉设计**：
  - 使用绿色调表示成功
  - Markdown 渲染（如适用）
  - 支持代码高亮

### 4. 原始响应 (Raw Response)
- **目的**：展示完整的 API 响应（用于调试）
- **展示位置**：详细视图标签页
- **视觉设计**：
  - JSON 格式化显示
  - 灰色调表示原始数据
  - 支持展开/折叠对象

### 5. 执行日志 (Logs)
- **目的**：记录时间线和事件
- **展示位置**：底部固定区域
- **视觉设计**：
  - 终端/控制台风格
  - 深色背景
  - 颜色编码级别：
    - 🔵 信息 (Info) - 绿色
    - 🟡 警告 (Warn) - 黄色
    - 🔴 错误 (Error) - 红色
  - 时间戳显示

### 6. 计时信息 (Timing)
- **目的**：显示性能指标
- **展示位置**：详细信息顶部
- **视觉设计**：
  - 开始/结束时间
  - 耗时高亮显示
  - 毫秒级精度

## 视图模式

### 简洁视图 (Simple View)
- 只显示 Markdown 渲染的输出
- 适合快速阅读和审查
- 隐藏技术细节

### 详细视图 (Detailed View)
- 标签页布局，可切换查看各类信息
- 适合调试和深入了解
- 显示所有技术细节

## 交互设计

### 标签页切换
```
[输入] [输出] [系统提示] [原始响应]
```
- 点击切换内容
- 当前激活标签高亮显示
- 图标 + 文字双重标识

### 复制功能
- 一键复制任何内容块
- 复制成功反馈

### 折叠/展开
- 长内容默认折叠
- 支持一键展开全部

## 组件结构

```
InfoPanel (主容器)
├── Tabs (标签页导航)
├── TimingInfo (计时信息)
└── ContentArea (内容区域)
    ├── InputTab
    ├── OutputTab
    ├── SystemPromptTab
    └── RawResponseTab
```

## 颜色方案

| 信息类型 | 背景色 | 边框色 | 文字色 |
|---------|--------|--------|--------|
| 系统提示 | #f5f3ff (violet-50) | #ddd6fe (violet-200) | #5b21b6 (violet-700) |
| 用户输入 | #eff6ff (indigo-50) | #bfdbfe (indigo-200) | #3730a3 (indigo-700) |
| AI 输出 | #f0fdf4 (green-50) | #bbf7d0 (green-200) | #15803d (green-700) |

## 动画效果

- 标签切换：淡入淡出
- 内容加载：骨架屏
- 新日志出现：滑入效果
- 错误提示：轻微抖动

## 响应式设计

- **移动端**：标签垂直排列，内容全宽
- **平板**：标签水平排列，内容两列
- **桌面**：充分利用空间，支持并排查看
